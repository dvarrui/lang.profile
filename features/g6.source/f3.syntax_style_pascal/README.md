

# Pascal syntax style 

* Los bloques de instrucciones entre BEGIN-END.
* Cada línea se separa de la anterior por punto y coma.

Lenguajes de programación con este estilo:
* Action!
* Ada
* Pascal

# Ejemplo 

Estructura de control condicional:
```
IF condition1 THEN
  BEGIN
    procedure1;
    procedure2
  END
ELSE
  BEGIN
    procedure3;
    procedure4
  END;
```
